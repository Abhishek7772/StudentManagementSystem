# Student Management System (Java)

A beginner-friendly console-based Java application built using Core Java concepts.

## Features
- Add student
- View all students
- Search student by ID
- Delete student

## Technologies Used
- Java
- Core Java (OOP, Collections, Scanner)
- IntelliJ IDEA

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `Main.java`
